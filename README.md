# Prompt Engineering 101

A comprehensive collection of resources and guides for mastering Prompt Engineering.

## Repository Structure

- `/OpenAI-Prompt-Guide`: Notes and resources based on the official OpenAI Prompt Engineering guide.
- `/LearnPrompting.org`: Exercises and documentation from LearnPrompting.org.

## Overview

This repository is dedicated to documenting the journey of learning and implementing advanced prompting techniques for Large Language Models (LLMs).
